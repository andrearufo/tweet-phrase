Tweet Phrase ===

A simple Wordpress plugin for tweet a single phrase with a click!

== Description == Using the shortcode [tweet] … [/tweet] allow your site users to click on a link that post the content of the shortcode on twitter. Really simple to use, with a modern style for the links and some options.

== Installation == Just download the plugin and put it on the /wp-content/plugins directory of you Wordpress installation.

== Frequently Asked Questions == * Is Twitter API or login required? No
